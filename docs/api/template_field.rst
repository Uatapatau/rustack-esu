TemplateField
=============

Каждый шаблон системы :class:`esu.Template` содержит список полей шаблона. Он
может быть получен с помощью метода :func:`esu.Template.get_fields`.

Поля специфичны для каждого из шаблонов и являются входными данными,
необходимыми для его успешного развертывания в виртуальный сервер.
